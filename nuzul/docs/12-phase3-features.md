# 12 — Phase 3 Features

Goal: scale, professional supply, and ecosystem.

## Marketplace scale
- **Dynamic pricing** suggestions (demand/seasonality/events).
- **Ads engine** (home banners, in-search placements) with targeting + reporting.
- **Host subscriptions** (Free/Pro/Agency) with tiered tools & lower fees.
- Loyalty, referrals, gift cards, promo codes.

## Professional supply
- **Property-manager / agency** consoles: multi-listing, team seats, channel manager.
- **Channel sync** to Booking.com/Airbnb via iCal/API.
- Bulk import & API for PMS integrations.

## Ecosystem & intelligence
- **Experiences/tours** marketplace; long-stay & corporate travel.
- **AI concierge** & smart search; fraud ML; review summarization.
- **Analytics warehouse** + host BI exports.

## Reach
- Native iOS/Android wrappers (Capacitor/Expo over the PWA core).
- **Maghreb expansion** (Tunisia, Morocco) — multi-currency, per-country config.
